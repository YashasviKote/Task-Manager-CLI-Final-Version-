# Task Manager CLI (Final Version)

## Description
A command-line based Task Manager application built using Java.  
This project demonstrates a complete backend-style system with CRUD operations, file persistence, input validation, and clean architecture.

---

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Search tasks by keyword
- View completed tasks
- Persistent storage using file system
- Input validation and error handling

---

## Concepts Covered
- Core Java
- Object-Oriented Programming (OOP)
- Collections Framework
- Streams API
- Optional
- File Handling (BufferedReader, BufferedWriter)
- Exception Handling
- CLI-based application design
